ðŸ¤– How to Connect Bots

Bots need to connect via network. Run this command on each bot machine:

python client_python.py SERVER_IP PLAYER [PORT]

Examples:

  • python client_python.py 192.168.1.100 X 8081
  • python client_python.py 192.168.1.100 O 8082

Requirements:

  • Install requests: pip install requests
  • Bots must be on the same network as the server
  • Each bot needs a unique port

ðŸ¤– Bot Protocol Documentation

Your bot must provide an HTTP endpoint at /get_move:

  • Method: POST
  • Input: JSON with game state
  • Output: JSON with move: {"row": 7, "col": 8}

Game state structure:

{
  "board": [15x15 array],
  "currentPlayer": "X" or "O",
  "gameOver": true/false,
  "winner": "X", "O", "Tie", or null
}

Example bot strategies:

  • Random: Pick any empty cell
  • Center: Always play in the center
  • Win First: Check for winning moves, then blocking moves
  • Pattern: Look for patterns and create lines

Board coordinates: (0,0) is top-left, (14,14) is bottom-right
